solutions_to_common_problems
=====

## Create new project

----	
	
	# all commands in one string
	$ rebar3 new lib solutions_to_common_problems && cd solutions_to_common_problems
	
## Get dependencies
	$ rebar3 deps	

## Format
	$ rebar3 format
	
## EUnit
-----
	$ rebar3 eunit
	$ rebar3 eunit -m common_problems_solutions_tests
